Rust Style Tips

  1. Use an accumulator to avoid repeated appending

  2. Don’t allow(unused)

  3. Don’t separate the check from the operation

  4. Functions should not unconditionally clone arguments

  5. Define an error type that includes the errors you care about

  6. Don’t repeat yourself in tests

  7. Don’t let error handling obscure the point

  8. Don’t “double buffer” your text formatting output

  9. If producing a sequence in order, return an iterator

  10. Strings are for text

  11. Avoid extra indirections in multidimensional arrays
